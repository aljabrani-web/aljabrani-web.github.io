# لوحة التحكم الإدارية - دليل الاستخدام

لوحة تحكم شاملة لإدارة موقعك الشخصي بسهولة ومرونة.

## 🚀 المميزات الرئيسية

### ✅ إدارة شاملة للمحتوى
- **المعلومات الشخصية**: تحديث الاسم، المسمى الوظيفي، النبذة، ومعلومات التواصل
- **المهارات التقنية**: إضافة وتعديل المهارات مع نسب الإتقان
- **معرض الأعمال**: إدارة المشاريع مع الصور والروابط والتصنيفات
- **المدونة**: محرر نصوص غني لكتابة وتعديل المقالات
- **الإعدادات**: تخصيص إعدادات الموقع والأمان

### 🔒 نظام أمان متقدم
- تسجيل دخول آمن مع تشفير كلمات المرور
- حماية من المحاولات المتكررة
- إدارة الجلسات الآمنة
- تغيير كلمة المرور

### 📱 تصميم متجاوب
- يعمل على جميع الأجهزة (كمبيوتر، تابلت، موبايل)
- واجهة مستخدم حديثة وسهلة الاستخدام
- دعم كامل للغة العربية

### 💾 إدارة البيانات
- حفظ تلقائي للبيانات
- تصدير واستيراد البيانات
- نسخ احتياطية
- تحديث الموقع الرئيسي تلقائياً

## 📁 هيكل الملفات

```
admin/
├── login.html              # صفحة تسجيل الدخول
├── dashboard.html          # الواجهة الرئيسية للوحة التحكم
├── admin-styles.css        # ملف التنسيق الرئيسي
├── admin-core.js           # النظام الأساسي للوحة التحكم
├── dashboard.js            # وظائف إدارة المحتوى
├── website-updater.js      # نظام تحديث الموقع الرئيسي
└── README.md              # دليل الاستخدام
```

## 🔧 التثبيت والإعداد

### 1. رفع الملفات
```bash
# ارفع مجلد admin إلى نفس مستوى الموقع الرئيسي
website/
├── index.html
├── styles.css
├── script.js
└── admin/
    ├── login.html
    ├── dashboard.html
    └── ...
```

### 2. الوصول للوحة التحكم
- اذهب إلى: `https://yourwebsite.com/admin/login.html`
- بيانات الدخول الافتراضية:
  - **اسم المستخدم**: `admin`
  - **كلمة المرور**: `admin123`

### 3. تغيير كلمة المرور
⚠️ **مهم**: غيّر كلمة المرور فور تسجيل الدخول الأول!

1. اذهب إلى قسم "الإعدادات"
2. في قسم "إعدادات الأمان"
3. أدخل كلمة المرور الحالية والجديدة
4. اضغط "تغيير كلمة المرور"

## 📖 دليل الاستخدام

### إدارة المعلومات الشخصية
1. اذهب إلى قسم "المعلومات الشخصية"
2. املأ النموذج بمعلوماتك
3. ارفع صورتك الشخصية
4. أضف روابط حساباتك على الشبكات الاجتماعية
5. اضغط "حفظ التغييرات"

### إدارة المهارات
1. اذهب إلى قسم "المهارات التقنية"
2. اضغط "إضافة مهارة جديدة"
3. اختر اسم المهارة والأيقونة
4. حدد نسبة الإتقان
5. اضغط "إضافة المهارة"

### إدارة معرض الأعمال
1. اذهب إلى قسم "معرض الأعمال"
2. اضغط "إضافة مشروع جديد"
3. املأ تفاصيل المشروع
4. ارفع صورة المشروع
5. أضف الروابط والتقنيات المستخدمة
6. اضغط "حفظ المشروع"

### إدارة المدونة
1. اذهب إلى قسم "المدونة"
2. اضغط "كتابة مقال جديد"
3. استخدم المحرر الغني لكتابة المحتوى
4. اختر التصنيف والكلمات المفتاحية
5. اختر "نشر المقال" أو "حفظ كمسودة"

### تحديث الموقع الرئيسي
1. بعد إجراء التغييرات، اذهب إلى "الإعدادات"
2. في قسم "تصدير البيانات"
3. اضغط "تحديث الموقع الرئيسي"
4. اتبع التعليمات المعروضة

## 🔄 تحديث الموقع الرئيسي

### الطريقة التلقائية
1. اضغط "تحديث الموقع الرئيسي" في الإعدادات
2. ستظهر نافذة مع التعليمات
3. حمّل ملف التحديث
4. ارفعه للموقع واتبع التعليمات

### الطريقة اليدوية
1. صدّر البيانات من قسم الإعدادات
2. استخدم البيانات لتحديث الموقع يدوياً

## 💾 النسخ الاحتياطية

### إنشاء نسخة احتياطية
1. اذهب إلى "الإعدادات"
2. في قسم "تصدير البيانات"
3. اضغط "تصدير البيانات"
4. احفظ الملف في مكان آمن

### استعادة نسخة احتياطية
1. اذهب إلى "الإعدادات"
2. في قسم "استيراد البيانات"
3. اسحب ملف النسخة الاحتياطية أو اختره
4. اضغط "استيراد البيانات"

## 🛡️ الأمان

### نصائح الأمان
- غيّر كلمة المرور الافتراضية فوراً
- استخدم كلمة مرور قوية (8 أحرف على الأقل)
- لا تشارك بيانات الدخول مع أحد
- أنشئ نسخ احتياطية بانتظام

### حماية المجلد
أضف ملف `.htaccess` في مجلد admin:
```apache
# حماية إضافية للوحة التحكم
<Files "*.html">
    # يمكن إضافة قيود IP هنا
</Files>
```

## 🔧 استكشاف الأخطاء

### مشاكل شائعة وحلولها

**لا يمكن تسجيل الدخول:**
- تأكد من صحة اسم المستخدم وكلمة المرور
- امسح ذاكرة التخزين المؤقت للمتصفح
- تأكد من تفعيل JavaScript

**لا تظهر التغييرات على الموقع:**
- تأكد من تحديث الموقع الرئيسي
- امسح ذاكرة التخزين المؤقت
- تحقق من رفع ملفات التحديث

**مشاكل في رفع الصور:**
- تأكد من أن حجم الصورة أقل من 5 ميجابايت
- استخدم صيغ الصور المدعومة (JPG, PNG, GIF)

### الحصول على المساعدة
- راجع هذا الدليل أولاً
- تحقق من console المتصفح للأخطاء
- أنشئ نسخة احتياطية قبل إجراء تغييرات كبيرة

### تطوير مخصص
يمكن تخصيص لوحة التحكم حسب احتياجاتك:
- إضافة أقسام جديدة
- تخصيص التصميم

## 📝 ملاحظات مهمة

- لوحة التحكم تعمل بـ JavaScript ولا تحتاج خادم
- البيانات محفوظة في localStorage المتصفح
- للاستخدام المتقدم، يُنصح بربط قاعدة بيانات
- النظام مصمم للاستخدام الشخصي وليس للمواقع الكبيرة
