5تفعيل GitHub Pages

بعد رفع جميع الملفات، فعّل GitHub Pages:

  1. اضغط تبويب "Settings" في المستودع
  2. انزل في القائمة الجانبية إلى "Pages"
  3. في Source، اختر "Deploy from a branch"
  4. اختر Branch: "main"
  5. اختر Folder: "/ (root)"
  6. اضغط "Save"
⚙️ فتح إعدادات Pages
